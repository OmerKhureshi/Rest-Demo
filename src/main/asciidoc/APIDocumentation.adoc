= RestDemo - RESTful  messenger web service.

== Introduction
This project demonstrates a web service to serve as provider for a basic messenger application. The end points allow GET and POST verbs to manage person and message entities. In addition, helpful queries can also be executed such as listed below.
1. Get list of messages by specifying sender, receiver or both.
2. Uses Spring HATEOS to provide useful links to navigate to related resources.

//In this project, there are two entities, Person and Message, as described below.
//=== Person
//.Entities
//|===
//|Field |Description
//
//|id
//|Unique identifier for a person. Id is auto-generated
//
//|firstName
//|A person's first name.
//
//|lastName
//|A person's last name.
//|===
//
//=== Message
//.Entities
//|===
//|Field |Description
//
//|id
//|Unique identifier for a person. Id is auto-generated
//
//|sender
//|Ref to person who sent the message.
//
//|receiver
//|Ref to person who received the message.
//
//|messageDesc
//|The textual message.
//|===
